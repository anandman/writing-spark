# Writing Spark ✨

A distraction-free "Oblique Strategies" deck for kids. It helps students overcome writer's block in Chinese and English writing assignments.

It creates **zero maintenance** for parents. There is no data entry required. The student simply opens the app and hits the button when they are stuck.

## Features

1.  **Chinese Mode (Sentence Expander):**
    *   Goal: Help students stretch simple sentences into 15+ character sentences.
    *   Sparks: "Add a color word", "Describe the sound", "Start with 'Suddenly'".
2.  **English Mode (Story Helper):**
    *   Goal: Help students sustain a narrative flow for 15+ sentences.
    *   Sparks: "Describe the smell", "Start with 'However'", "What is the character thinking?".

## How to Use

1.  **Open the App:** Visit the webpage on an iPad, Chromebook, or Phone.
2.  **Select Mode:** Choose "Chinese Sentence" or "English Story".
3.  **Get an Idea:** Tap the big **"✨ Give me an Idea"** button whenever stuck.

## Deployment

This project is hosted on **GitHub Pages**.

1.  Upload `index.html` and `prompts.js` to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch as the source.
4.  Your app will be live at `https://[username].github.io/[repo-name]/`.

## Development

*   `prompts.js`: The database of creative strategies. Edit this file to add more nudges.
